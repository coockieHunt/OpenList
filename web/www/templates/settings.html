À propos d'OpenList

Une application ultra-légère de liste de courses.

Philosophie du projet

Le but d'OpenList est d'offrir une gestion de listes performante en utilisant le moins de ressources possible. L'application est optimisée pour la rapidité et une empreinte disque minimale.

Taille estimée de l'exécutable : ~10-15 Mo

Stack Technique

  • Backend : Go (Lang) pour l'API haute performance.
  • Serveur Web : Mini-serveur intégré en Go.
  • Base de données : SQLite (fichier local unique).
  • Frontend : HTML5 & Pico.css (Lightweight CSS).
  • Icônes : SVG optimisés de Tabler Icons.
